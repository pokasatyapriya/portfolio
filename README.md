# Satya Priya Poka — AI/ML Engineer Portfolio

🌐 **Live Site:** [pokasatyapriya.github.io/portfolio](https://pokasatyapriya.github.io/portfolio)

A modern, dark-themed single-page portfolio built with pure HTML, CSS, and JavaScript.

## Sections
- 🏠 Hero — Name, title, animated stats, resume download
- 👤 About — Bio, contact info, spinning avatar animation
- 🧠 Skills — 6 categorized skill cards with tech tags
- 💼 Experience — Vertical timeline (Cardinal Health, Wells Fargo, CTS)
- 🏆 Certifications — AWS, TensorFlow, Azure, Databricks, CKAD
- 🎓 Education — MS (USM) + B.Tech (Vignan's)
- 📬 Contact — Email, Phone, LinkedIn, Resume download

## Features
- Glassmorphism navbar with scroll-based active link highlighting
- Animated floating particles in hero
- Fade-in scroll animations on all cards
- Fully responsive (mobile-friendly)
- Resume download button wired to `.docx` file

## Tech Stack
- Pure HTML5 + CSS3 + Vanilla JS (no frameworks)
- Google Fonts (Inter)
- Font Awesome 6 icons

## Deployment
Hosted via **GitHub Pages** from the `main` branch.
